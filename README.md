# PreEmulation
A ***(Super early expermental)*** emulator for the PreComputer line of computers.


Computers:
   - PreComputer 1000 - testing (variations are untested)

Parts:
   - CPU:
  	 - Z80 - Working

   - IO:
  	 - IO-LCD - skeleton/testing
  	 - IO-Keyboard - None

   - Video/LCD:
 	  - HD44780:
		 - DDRAM - Basics
		 - CGRAM - none
		 - Control-funtions - Unknown
		 - Byte-funtions - Unknown
		 - Byte-funtions - Unknown
		 - Video - none
